Taught by [Christoph Spiegel](https://christophspiegel.berlin) (ZIB) with [Silas Rathke](https://www.mi.fu-berlin.de/en/math/groups/geokomb/People/PhD-Students/Silas_Rathke.html) (FUB) as teaching assistant.
